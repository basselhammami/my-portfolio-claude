Balady+ Maps — Navigation Redesign

Redesigning the navigation experience for Saudi Arabia's official government maps application.

ROLE
Middle Product Designer
COMPANY
Balady+
TIMELINE
Oct 2023 – Sep 2024
DOMAIN
Mobile / Government

Introduction

Balady+ is Saudi Arabia's official government maps and municipal services application, used by millions of citizens to navigate cities, report issues, and access government services.

My role focused on redesigning the core navigation experience — specifically the navigation mode and the introduction of a new free roam feature that allowed users to explore the map without a fixed destination.

The Challenge

Three interconnected problems defined the navigation experience:

01

Navigation mode friction

The existing turn-by-turn navigation had high cognitive load, with too many elements competing for attention during active driving.

02

No exploration mode

Users who wanted to explore an area without a specific destination had no way to do so — the app required a destination to start.

03

Horizontal navigation gap

The app lacked a horizontal navigation pattern for browsing categories and services, forcing users into deep menu hierarchies.

These weren't isolated UI problems — they reflected a fundamental mismatch between how users wanted to interact with the map and how the app was designed.

Service Blueprint

Before designing solutions, I mapped the complete user journey across all navigation scenarios to identify where the experience broke down.

Service blueprint — full navigation journey

The Solution

Three interconnected features that addressed the core navigation problems identified in discovery.

Navigation Mode Redesign

A simplified navigation interface that reduces cognitive load during active driving by surfacing only the information needed at each moment.

Free Roam Mode

A new exploration mode that allows users to browse the map without a fixed destination, enabling discovery and spontaneous navigation.

Horizontal Navigation

A horizontal scrolling pattern for browsing service categories, reducing menu depth and making discovery significantly faster.

Feature Details

Each feature was designed to address a specific pain point while integrating seamlessly with the existing map experience.

Parking Enhancements

Finding and navigating to parking was one of the most common use cases in urban areas. The redesigned parking feature surfaces real-time availability and integrates payment directly in the flow.

The parking overview screen shows nearby options with availability, pricing, and distance — all without leaving the map.

Parking overview & detail view

The detail view provides real-time space availability, operating hours, and direct navigation with one tap.

Free Roam Mode

A new exploration mode that allows users to browse the map without a fixed destination. Users can explore areas, discover points of interest, and start navigation from any point.

The transition between free roam and navigation mode is seamless — tap any point of interest to get directions, or tap the navigation button to start from your current location.

Free Roam — map controls

Horizontal Navigation

A horizontal scrolling navigation pattern for browsing categories and services, reducing the depth of menu hierarchies and making discovery faster.

Horizontal navigation — category browsing

Impact

Reduced navigation friction

User testing showed a significant reduction in navigation errors and mid-journey cancellations with the redesigned navigation mode.

New exploration behaviour

Free roam mode introduced a new usage pattern — users spending more time exploring the map without a specific destination.

Faster service discovery

Horizontal navigation reduced the average number of taps to reach a service category from 4 to 2.
